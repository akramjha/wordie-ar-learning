# Wordie AR

## Overview

Wordie AR is an augmented reality mobile application developed using Unity and Vuforia. The application allows users to explore planets and space-related content through interactive AR experiences.

## Features

- Augmented Reality Visualization
- Interactive Planet Exploration
- 3D Object Rendering
- Educational Learning Content
- Mobile-Friendly Interface

## Technologies Used

- Unity
- C#
- Vuforia Engine
- Android

## Skills Demonstrated

- AR Development
- Mobile Application Development
- Unity Development
- User Experience Design
- Educational Technology
