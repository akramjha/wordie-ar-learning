# Wordie AR Learning

## Overview

Wordie AR Learning is an educational Augmented Reality (AR) mobile application developed using Unity and Vuforia. The application introduces children to basic learning concepts through interactive 3D models.

By scanning image targets, children can view 3D objects and learn about different categories such as animals, fruits, and shapes in a more engaging way.

---

## Key Highlights

- Developed using Unity and Vuforia Engine
- Marker-based Augmented Reality application
- Interactive 3D educational content
- Child-friendly learning experience
- Android mobile application

## Application Workflow

1. Launch the application
2. Open AR camera
3. Scan image target
4. Display 3D object
5. Explore animals, fruits, or shapes
6. Continue learning with different markers
